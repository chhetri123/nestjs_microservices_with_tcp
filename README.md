# Nestjs microservices with TCP

## Description

 This is an example repo of how microservice works on basics level. How it communicate like server and  client way. 

## Installation for nest_microservice

```bash
$ cd nest_microservice
$ npm install
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
